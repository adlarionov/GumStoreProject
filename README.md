<h1 align="center"> START PROJECT </h1>

<h3> Install All Dependencies </h3>

<code>
cd backend <br>
npm install <br>
npm run build <br>
<br>
cd frontend <br>
npm install <br>
</code>

To backend you need to add .env file with random <span>APP_KEYS <span>

<hr />
<h3>Start Project</h3>
<code>
cd backend <br>
npm run develop <br>
<br>
cd frontend <br>
npm start <br>
</code>
